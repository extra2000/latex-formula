## Supported platforms

| OS | Architecture |
| -- | ------------ |
| Fedora 32, 33 | `x86_64` |
| Arch Linux | `x86_64` |
| OpenSUSE Leap 15.2 | `x86_64` |
| OpenSUSE Tumbleweed | `x86_64` |
| Ubuntu 20.04 | `x86_64` |

Note: `aarch64` platforms are not yet supported.
